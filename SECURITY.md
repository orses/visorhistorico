# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it **privately** using GitHub's built-in reporting tool:

1. Go to the **Security** tab of this repository
2. Click **"Report a vulnerability"**
3. Describe the issue and its potential impact

Do **not** open a public issue for security vulnerabilities.

We will acknowledge the report and respond as soon as possible.
